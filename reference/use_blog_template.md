# Create folder and files for website blog post

Create folder and files for website blog post

## Usage

``` r
use_blog_template(date_chr = "2025-12-31", slug)
```

## Arguments

- date_chr:

  Date in yyyy-mm-dd format.

- slug:

  Character string with url slug e.g. `"blog-post-title"`

## Value

a message if template file was successfully copied over
